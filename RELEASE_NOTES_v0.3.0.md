# Release Notes - v0.3.0

## New Features

- **AI Bot Client Mode**: Added "AI Bot Client" button to the title screen. Input target LAN IP:port to auto-connect.
- **AI Auto-Control**: After joining, AI perceives the environment every ~3 seconds and autonomously decides actions: move, jump, sneak, attack, use item, chat.
- **Environment Perception**: AI senses position, health, hunger, nearby entities (hostile/passive/players), time of day.
- **Auto Chat Reply**: AI automatically replies when mentioned in chat.
- **Auto Attack**: AI turns toward and attacks nearby hostile mobs.

### Compatibility

- Minecraft 1.21.1
- Fabric Loader >= 0.16.9
- Fabric API 0.102.1+1.21.1
- Java >= 21
