# mc-1.21.1-ai-companion-mod

Minecraft 1.21.1 Fabric 客户端 Mod，用于在游戏中查看已加载地图，并配置本地或云端 AI 同伴。

## 功能

- 按 `L` 键打开已加载地图界面。
- 已加载区块显示为绿色，未加载区块显示为黑色，玩家所在区块显示为黄色。
- 在按 `ESC` 后的游戏菜单中增加“加入本地或云端AI”按钮。
- AI 设置界面支持“云端”和“本地”两种入口。
- 云端入口支持添加 API 和模型名。
- 本地入口支持 `ollama` 与 `llama.cpp` 两个服务商。
- `ollama` 默认连接 `127.0.0.1:11434`。
- `llama.cpp` 默认连接 `127.0.0.1:8080`。
- 配置完成后可选择“聊天”或“加入游戏”。
- 聊天模式和加入游戏模式都支持在游戏内输入 `@*模型名` 或 `@模型名` 与模型对话。

## 构建

```powershell
gradle build
```

需要 Java 21。首次构建会从 Fabric Maven 下载依赖。

## 说明

当前版本已实现客户端 UI、配置保存、地图显示、本地 `ollama` / `llama.cpp` HTTP 调用入口和聊天拦截。云端 API 已预留输入与路由位置，不同云端服务商的请求格式需要在 `AiRouter.cloudReply` 中按实际协议补充。
